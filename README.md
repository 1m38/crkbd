# crkbd
Corne kbd, a split keyboard with 3x6 vertically staggered keys and 3 thumb keys, based on [Helix](https://github.com/MakotoKurauchi/helix) and [MiniDox](https://github.com/That-Canadian/MiniDox_PCB).
